# Nexus-Code (React Client)

This repository contains the v2 (React) implementation of the Nexus-Code project. 

## Why React? (Migration History)
The initial prototype (v1) was built using Vanilla JavaScript and HTML. As the project grew, direct DOM manipulation became harder to manage. To improve component reusability, state management, and overall performance for the collaborative workspace features, the architecture has been migrated to React.

* **Tech Stack:** React, Vite, Tailwind CSS v4
